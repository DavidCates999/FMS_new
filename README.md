# FMS Query Engine

AI-Powered Franchise Management Analytics - Transform natural language into powerful database insights.

## Features

- 🔐 Role-based authentication (Super Admin, System Admin, Franchise Partner, Client Admin)
- 🏢 Franchise-level data isolation
- 🤖 AI-powered natural language to MongoDB query conversion
- 📊 Real-time data visualization
- 📥 Export to CSV/JSON

## Technology Stack

- **Frontend:** Streamlit
- **Database:** MongoDB
- **AI:** OpenAI GPT-4 / Anthropic Claude
- **Deployment:** Docker on Hugging Face Spaces
